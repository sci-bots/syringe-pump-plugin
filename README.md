Syringe Pump Plugin
===================

This is a [MicroDrop][1] plugin for interfacing a syringe pump with the [DropBot][2] Digital Microfluidics control system.

[1]: http://microfluidics.utoronto.ca/microdrop
[2]: http://microfluidics.utoronto.ca/dropbot
